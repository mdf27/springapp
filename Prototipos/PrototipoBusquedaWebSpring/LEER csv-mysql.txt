CONSOLA

1) abrir consola en modo administrador.

2) ir a la carpeta bin de MySQL

3) ejecutar: mysql -h localhost -u root -p 
Si les pide contraseña en mi caso es vacía.

4) ejecutar: create database prueba_lucene

5) ejecutar: use prueba_lucene 

6) ejecutar:
create table `productos` (
`nro_articulo` bigint(10) default '0',
`lab` char(10) default null,
`lin` char(10) default null,
`clave1` char(10) default null,
`clave2` char(10) default null,
`clave3` char(10) default null,
`descripcion` char(30) default null,
`complemento_des` char(30) default null,
`viene_barra` char(5) default null,
`codigo_barra` char(20) default null,
`porc_contado` bigint(10) default null,
`porc_oferta` bigint(10) default null,
`habilitado` char(5) default null,
`precio_venta` bigint(10) default null,
`precio_oferta` bigint(10) default null,
`t_iva` bigint(5) default null
) ENGINE=InnoDB DEFAULT CHARSET=latin1;


7) El archivo stock que les dio el cliente está en formato xls, guardarlo en formato csv

8) guardar el archivo stock.csv en la carpeta mysql>data>prueba_lucene

9) ejecutar:
LOAD DATA INFILE 'stock.csv'
INTO TABLE productos
FIELDS TERMINATED BY ','
LINES TERMINATED BY '\n'
(nro_articulo,lab,lin,clave1,clave2,clave3,descripcion,complemento_des,viene_barra,codigo_barra,porc_contado,porc_oferta, t_iva);
